Wordle spel dokumentation och regler

Skoluppgift där vi ska bygga en egen version av ett wordle spel:

  • Skapa ett wordle spel med React där delar av logiken byggs på backend
  • Server side rendera en highscorelista som dynamiskt ändras när användaren spelar och får ett resultat

Spelregler

  1. välj dina inställningar: antal bokstäver och tillåt upprepning av bokstav eller ej
  2. Gissa ett ord i textrutan
  3. Nu ser du ordet nedanför textrutan med antingen röda, gula eller gröna bokstäver
  4. grön = bokstaven finns på rätt plats
    gul = bokstaven finns med i ordet men är på fel plats
    röd = bokstaven finns inte med i ordet
